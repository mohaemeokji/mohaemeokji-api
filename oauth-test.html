📌 카카오 로그인 테스트 방법

방법 1: 카카오 개발자 콘솔에서 토큰 발급 (가장 간단!)

  • 카카오 개발자 콘솔 > 토큰 메뉴
  • "카카오 계정으로 로그인" 버튼 클릭
  • 표시된 "액세스 토큰" 전체 복사
  • 아래 입력란에 붙여넣고 테스트

🔗 바로 가기: 카카오 개발자 콘솔

⚠️ 토큰 발급 방법

방법 1: 카카오 API 테스트 도구 (권장)
1. 카카오 API 테스트 도구 접속
2. 우측 상단 "로그인" 클릭
3. 카카오 계정으로 로그인
4. "액세스 토큰" 복사 (하단에 표시됨)
5. 아래에 붙여넣기

⚠️ 주의사항:
- 카카오 개발자 콘솔에서 앱을 먼저 생성하세요
- 동의 항목에서 이메일을 "필수 동의"로 설정하세요
- 토큰은 약 6시간 동안 유효합니다

⚙️ 설정

위 링크에서 토큰을 복사하여 여기에 붙여넣으세요

📌 애플 로그인 테스트 방법

네이티브 앱에서 Identity Token 받기

  • iOS/Android 앱에서 애플 로그인 실행
  • Identity Token 복사
  • 아래 입력란에 붙여넣기
  • "테스트" 버튼 클릭

⚠️ 애플 로그인 제한사항

애플 로그인은 웹에서 테스트가 제한적입니다.
네이티브 앱(iOS/Android)에서 Identity Token을 받아 테스트하세요.

웹에서 테스트하려면: Apple Developer 계정, Services ID 등록, 도메인 검증이 필요합니다.
